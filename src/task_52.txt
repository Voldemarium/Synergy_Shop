Урок 13. SortedMap / NavigableMap / TreeMap
Цель задания:
Познакомиться с интерфейсами SortedMap / NavigableMap, а также с классом TreeMap, а также сформировать базовые навыки
по работе с ними

Задание:
Вам предстоит сделать онлайн магазин, без графического интерфейса, обладающего следующими функциями:
●Каталог товаров
●Авторизация почта и пароль (данные возможно хранить в простом незашифрованном виде)
●Реализовать вывод каталога
●Добавление товара по ID + количество товара
●Возможность очистки корзины, просмотр корзины, оформление заказа с вводом данных о пользователе

3 слоя:
------------
dataSources  - взаимодействие с БД
------------
services     - взаимодействие с dataSources
------------
controllers  - обработка введенных пользователем данных и команд
views        - взаимодействие c пользователем (показ в консоли)
------------
